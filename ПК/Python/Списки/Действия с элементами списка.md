pop()

pop(i)  удаляет выюранный элемент из списка 
pop() удаляет последний элемент из списка, но позволяет работать с ним
после удаления.
Пример
	world_list = ['America', 'Russia', 'China']  
	new_world = world_list.pop()  
	print(new_world)
__________________________________
